Registration Form HTML/CSS

Files
- index.html — the form markup
- styles.css — styles to replicate the attached image

How to use
1. Open [index.html](index.html) in your browser (double-click or drag into browser).
2. Fill the fields; to print, use the browser Print dialog (Ctrl+P) — layout is print-optimized.

Notes
- This is plain HTML/CSS (no server). Edit labels or fields directly in `index.html`.
- If you want a PDF, open the print dialog and choose "Save as PDF".
